# SuperStore Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using the SuperStore Sales dataset. The dashboard provides insights into sales performance, profit trends, customer segments, shipping modes, product categories, and regional sales. It also includes a 15-day sales forecast to support business decision-making.

## Tools Used
- Power BI
- Excel

## Dashboard Features

### Page 1: Sales Analysis Dashboard
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Shipping Days
- Sales by Segment
- Sales by Category
- Sales by Sub-Category
- Sales by Ship Mode
- State-wise Sales Analysis
- Monthly Sales Trend
- Monthly Profit Trend

### Page 2: Sales Forecast Dashboard
- Historical Sales Analysis
- 15-Day Sales Forecast
- State-wise Sales Comparison
- Trend Analysis

## Key Insights
- Consumer segment contributes the highest sales.
- Technology category generates significant revenue.
- Standard Class is the most frequently used shipping mode.
- California records the highest sales among all states.
- Sales forecasting helps identify future business trends.
